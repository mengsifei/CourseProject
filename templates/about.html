About this project

This project is Sifei Meng's third course project in AMI CS HSE.

The goal of the site is to get the best "face" frame of the video that can be the thumbnail of the video.

  • Num of frames: the number of frames to be processed. By default, the program will process take first 4000 frames of the video. The execution time depends on the size of the video. This variable can limit the execution time if the user wants to pick the best frame only from the first few frames. In order to take all frames, please enter 0.

  • Frame size: the width of the final frame. The height will also be adjusted while preserving the image proportion. By default, the site generates the best frame with an image width of 450px.
